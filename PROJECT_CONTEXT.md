# Project Context

Project Name: End-to-End MLOps Pipeline for Customer Churn Prediction on AWS

Goal:
Build a portfolio-ready MLOps project that trains a customer churn prediction model locally, serves it with FastAPI, containerizes it with Docker, pushes the image to AWS ECR, deploys it on AWS EC2, and automates deployment using GitHub Actions CI/CD.

Dataset:
Telco Customer Churn dataset with 7,043 customer records. Features include demographics, account information, services, and target variable Churn.

Target:
Churn column with values Yes/No.

Tech Stack:
Python, Pandas, Scikit-learn, FastAPI, Docker, AWS ECR, AWS EC2, GitHub Actions.

Development Style:
Work step by step. Do not build everything at once. Start with data inspection and model training.
